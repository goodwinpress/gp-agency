== Описание ==

Новая, красивая, быстрая тема от Гудвина для создания сайта по оказанию услуг, например, дизайн-агентства. 

Адаптивность, приятный внешний вид, готовые блоки на Главной, высокая скорость загрузки, дополнительные страницы для витрины и карточек проектов, услуг, контактов. Получение заявок через поп-ап и формы Contact Form 7. Встроенная кнопка для чатов под популярные мессенджеры. Микроразметка. Отложенная загрузка изображений и видео. Блог.
